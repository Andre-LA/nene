Nene's scancodes copies the values and names SDL's scancodes, with three differences:
* the `SDL_SCANCODE_` prefix it's removed.
* Intead of using an UPPERCASED identifier, only the first letter of the identifier it's capitalized
* Any number `n` uses `Num_` prefix (`SDL_SCANCODE_0` -> `Num_0`)

Related SDL documentation:
* [SDL_Scancode](https://wiki.libsdl.org/SDL_Scancode)
